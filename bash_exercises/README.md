# Bash Exercises

This directory contains educational bash shell scripts and exercises for learning shell scripting.

## Contents

### Basic Scripts
- **hello_world.sh** - Hello World introduction script
- **hello_test.sh** - Hello World with test framework

### Variable & Data Operations
- **print_numbers.sh** - Print number sequences
- **show_date.sh** - Display date and time information

### File Operations
- **list_files.sh** - List files in current directory
- **count_lines.sh** - Count lines in files
- **make_directory.sh** - Create directories
- **find_word.sh** - Search for words in files

### Advanced Operations
- **simple_calculator.sh** - Basic arithmetic calculator
- **test_custom_script.sh** - Test framework for custom scripts

## Purpose

These scripts serve as:
- **Learning resources** - Examples of common bash patterns
- **References** - Quick lookups for bash syntax and techniques
- **Exercises** - Progressively more complex shell scripting tasks

## Usage

### Running Scripts
```bash
chmod +x *.sh
./script_name.sh
```

### Learning Path

Start with basic scripts and progress to more complex ones:
1. `hello_world.sh` - Basics
2. `print_numbers.sh` - Variables and loops
3. `list_files.sh` - File operations
4. `simple_calculator.sh` - Advanced operations
