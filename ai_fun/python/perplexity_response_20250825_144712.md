# Perplexity API Response

## Prompt
```bash
Can you provide me a basic table in markdown?
```## Response
```
Here is a **basic Markdown table**:

```markdown
| Name  | Age | City      |
|-------|-----|-----------|
| Alice | 25  | New York  |
| Bob   | 30  | London    |
```

When rendered, it appears as:

| Name  | Age | City      |
|-------|-----|-----------|
| Alice | 25  | New York  |
| Bob   | 30  | London    |

This structure uses pipes (|) to separate columns and hyphens (-) to define the header row[3][1][5].
