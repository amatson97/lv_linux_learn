# Perplexity API Response

## Prompt
```
Can you provide me a basic table in markdown?
```
## Response
```
A basic Markdown table consists of a header row, a separator row, and one or more data rows, with columns separated by pipe (|) characters and the header separated by hyphens (-)[1][3][5].

Here is a simple example:

```markdown
| Name  | Age | City      |
|-------|-----|-----------|
| Alice | 25  | New York  |
| Bob   | 30  | London    |
```

This will render as:

| Name  | Age | City      |
|-------|-----|-----------|
| Alice | 25  | New York  |
| Bob   | 30  | London    |

- The **first row** contains the column headers.
- The **second row** uses hyphens to separate the header from the data.
- The **data rows** follow below, each separated by a newline[3][5].
